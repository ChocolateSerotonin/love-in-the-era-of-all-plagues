《霍乱时期的爱情》最打动我的是主人公漫长的人生轨迹：人在不同的人生阶段会如何理解爱情，又会如何因为时间、经历和选择而发生改变。这些变化很难用一句 Prompt 描述，却又真实存在于优秀的文学作品中。

这个vibe coding项目尝试把《霍乱时期的爱情》中的一个核心主题——从世俗个体走向本质之爱的过程——抽象成一个可以计算、可以模拟的模型。我将“执着度”“人生波动”“纪律性”等因素设计为可调参数，用不同的算法生成不同的人生轨迹，并通过路径的变化自动识别“顿悟”“低谷”“成长”等叙事节点，希望让故事在模型运行过程中自然涌现。

整个项目使用 Python 与 Streamlit 开发完成.它既是一件关于文学的实验，也是一次关于 AI 创作边界的探索：如果未来 AI 真正能够创作优秀的文学作品，它需要学习的也许不仅是语言本身，更是那些隐藏在作品背后的结构、节奏和人性。
